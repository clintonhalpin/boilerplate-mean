# Boilerplate MEAN

A starter template for a mean stack. Check the issues for future functionality.

# Getting Started

```shell
# Install Dependecies
npm i

# Start API
cd api
node server
```

# Current API Routes

| VERB  | URL |
| ------------- | ------------- |
| GET  | /api/items  |
| GET  | /api/items/:id  |
| POST  | /api/items/  |
| PATCH | /api/items/:id  |
| PUT  | /api/items/:id  |
| DELETE  | /api/items/:id  |
