# Boilerplate MEAN

A starter template for a mean stack. Check the issues for future functionality.

# Getting Started

```shell
# Install Dependecies
npm i

# Start API
cd api
node server
```

# Current API Routes

| VERB  | URL |
| ------------- | ------------- |
| GET  | /items  |
| GET  | /items/:id  |
| POST  | /items/  |
| PATCH | /items/:id  |
| PUT  | /items/:id  |
| DELETE  | /items/:id  |
